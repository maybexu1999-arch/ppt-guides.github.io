每一页动手前，先想3个问题
STEP 0
Q1
这页核心是什么？
一句话说清楚这页要干什么：
→ 一句结论 → 一个流程 → 几个并列方案 → 几组数据 → 一个对比 核心决定这页的「主角」是什么。
Q2
信息量多还是少？
字多和字少，完全不同的策略：
→ 字多：分区、建结构 用标签压缩视觉密度，让读者眼睛有锚点 → 字少：放大、留白 靠字本身撑气场，主标题放大到「觉得太大」才对
Q3
信息之间什么关系？
关系决定版式结构：
→ 并列 → 等宽卡片网格 → 递进/流程 → 横向箭头 → 从属 → 大框套小框 → 对比 → 左右各半+颜色区分
核心原则：版式是信息关系的视觉翻译。先理清信息关系，再选版式——而不是先找好看的版式再填内容。
